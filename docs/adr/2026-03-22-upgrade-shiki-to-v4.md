# Upgrade Shiki to v4

## Status

accepted

## Context

Shikiはシンタックスハイライトライブラリで、プロジェクトではMDX記事内のコードブロックのハイライト表示に使用している。Shiki v3を使用していたが、v4がリリースされた。

## Decision

Shikiを **v3からv4へメジャーアップグレード** する。

### アップグレード対象パッケージ

- `shiki`: 3.23.0 → 4.0.2
- `@shikijs/langs`: 3.22.0 → 4.0.2
- `@shikijs/rehype`: 3.22.0 → 4.0.2
- `@shikijs/themes`: 3.22.0 → 4.0.2

### Considered Options

1. **v4へアップグレード** (採用)
   - 最新の機能とパフォーマンス改善
   - 継続的なサポートとバグ修正

2. **v3のまま維持**
   - 安定性を優先
   - 既存コードへの影響なし

3. **他のシンタックスハイライトライブラリへ移行**
   - Prism.js, highlight.js等
   - 移行コストが高い

### 選択理由

<!-- TODO: 以下の選択理由を詳細化してください -->
- メジャーバージョンアップだが、破壊的変更が少ない（推測）
- rehypeプラグインのAPIに大きな変更がない
- 継続的なサポートとセキュリティアップデート
- パフォーマンス改善の恩恵
